# 环境安装与启动

## 1. 前置要求

- macOS / Linux
- Python `3.11.x`

说明：`open-webui` 当前发布版本对 Python 有上限约束。实测 Python `3.13` 无法安装，需要使用 `3.11`。

补充：如果你说的“openwebui1”是指 pip 安装版本，实际包名是 `open-webui`。

## 2. 创建 venv

```bash
python3.11 -m venv .venv
source .venv/bin/activate
python --version
```

## 3. 安装 OpenWebUI

```bash
pip install -U open-webui
```

验证：

```bash
open-webui --help
open-webui serve --help
```

## 4. 启动服务

```bash
source .venv/bin/activate
open-webui serve --host 127.0.0.1 --port 8080
```

访问：`http://127.0.0.1:8080`

## 5. 常见问题

### 5.1 Python 版本不兼容

现象：`No matching distribution found for open-webui`

处理：确保在 `3.11` venv 中执行安装。

### 5.2 首次启动慢

首次启动可能涉及：
- 数据库 migration
- 依赖资源下载（如模型文件）

属于正常行为，等待完成后重启会快很多。

### 5.3 CORS 警告

默认日志可能提示 `CORS_ALLOW_ORIGIN='*'` 不建议生产使用。

开发阶段可先保持默认，生产部署时再按白名单收紧。
